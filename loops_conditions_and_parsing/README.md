# 4. If 9 Say Hi

This Bash script displays **"Best School"** 10 times using a `while` loop.  
On the 9th iteration, it also prints **"Hi"** on a new line.  
It demonstrates the use of `while` loops together with `if` statements in Bash scripting.

## Usage

1. Make the script executable:

```bash
chmod +x 4-if_9_say_hi
